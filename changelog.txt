version 1.00

Первая сборка телеграмм бота. Основной внедренный функционал:
- ветка с события: внесение событий, просмотр событий
- кнопка для просмотра погоды по api. с краткой информацией о погоде в Мосвкве.

В docker контейнере развернута база и pgadmin.
Запись пользователей в базу при первичном входе.

Изменена кнопка стартового меню:
- есть кнока /start - вызывает стартовое меню
- есть кнопка /help - будет описан функционал бота, на данный момент неинформативно

Внесение событий, вносится информация в следующем порядке:
- user_id
- start_dt - Дата начала события
- end_dt - Дата окончания события
- name - Название события
- event_time - Время проведения события
- mode - Тип события
- place - Место проведения события
- description - Описание события (необязательно)

После внесения события предлагается команда /showdata, которая показывает последнее внесенное событие.

Просмотр событий:
- выдает список событий
- PDF с полным описанием всех параметров событий
Данные отдаются по user_id, только временные события.

Инфраструктурно:
- настроено логирование через logging
- для временного хранения используется MemoryStorage() aiogram
